# GitHub Cheat Sheet

A quick reference guide for essential Git and GitHub commands. This repository provides a centralized location for cheatsheets to improve workflow efficiency.

## Contents

- **Git Cheat Sheet from GitHub.pdf**: The official visual guide to common Git commands including:
  - Repository Initialization
  - Staging and Committing
  - Branching and Merging
  - Synchronizing with Remotes
  - Reverting Changes

## Quick Reference Commands

| Command | Description |
|---------|-------------|
| `git clone [url]` | Clone a remote repository |
| `git checkout -b [name]` | Create and switch to a new branch |
| `git pull` | Update your local repository with the latest changes from GitHub |
| `git push` | Upload your local changes to GitHub |
| `git log` | View the commit history |
